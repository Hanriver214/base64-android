# Base64 Android App

一个简单的 Android Base64 编码/解码工具应用。

## 功能特性

- Base64 编码：将任意文本编码为 Base64 格式
- Base64 解码：将 Base64 格式文本解码为原始文本
- 复制结果：一键复制输出结果到剪贴板
- 清空输入：快速清空所有内容
- 完全离线：不需要任何网络权限
- 零权限：不请求任何敏感权限
- 主题跟随系统：自动适配深色/浅色模式

## 技术特点

- 使用 Material Design 3 设计语言
- 支持 Android 8.0+ (API 26+)
- 最小化应用体积
- 完全离线运行

## 开发环境

- Android Studio Hedgehog | 2023.1.1
- Gradle 8.0
- Android SDK 34
- Java 17

## 构建

```bash
./gradlew assembleRelease
```
